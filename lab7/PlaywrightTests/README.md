# Playwright UI Automation Framework (C# / NUnit)

Цей проєкт є автоматизованим фреймворком для UI-тестування, побудованим з використанням **C#**, **Microsoft Playwright** та **NUnit**. Архітектура базується на патерні **Page Object Model (POM)**.

## Структура проєкту
- `Config/` - конфігураційні файли (JSON) та менеджер налаштувань.
- `Pages/` - класи сторінок (POM), що інкапсулюють локатори та методи взаємодії.
- `Tests/UI/` - тестові класи та налаштування базового середовища (BaseTest).
- `Utils/` - допоміжні утиліти (наприклад, кастомний Logger).

## Вимоги (Prerequisites)
1. Встановлений [.NET SDK](https://dotnet.microsoft.com/download).
2. Встановлені браузери для Playwright.

## Налаштування та запуск
1. Клонуйте репозиторій або відкрийте папку проєкту.
2. Встановіть необхідні браузери (якщо ще не встановлено):
   ```bash
   playwright install
